Diseño Gráfico

El diseño gráfico es la disciplina que se encarga de crear y proyectar conceptos a través de imágenes y texto. Un diseñador gráfico es el encargado de transformar ideas en soluciones visuales atractivas y efectivas.


  • Diseño de logotipos e identidad corporativa: Creación de la imagen visual de una marca.
  • Diseño editorial: Maquetación de libros, revistas, folletos, etc.
  • Diseño web: Creación de diseños atractivos y funcionales para páginas web.
  • Diseño de infografías: Visualización de datos de manera clara y atractiva.
  • Diseño de ilustraciones: Creación de ilustraciones personalizadas.

    • Diseño UX/UI

    El diseño UX/UI se enfoca en la experiencia del usuario al interactuar con un producto digital.


  • UX (User Experience): Se encarga de la experiencia completa del usuario, desde la investigación hasta la evaluación del producto.
  • UI (User Interface): Diseña la interfaz visual de un producto, haciendo que sea intuitiva y fácil de usar.


  • Diseño de interfaces de usuario: Creación de interfaces para aplicaciones móviles y web.
  • Diseño de wireframes y prototipos:Creación de modelos iniciales de una interfaz.
  • Testing de usabilidad:Evaluación de la facilidad de uso de un producto.

    • Desarrollo en Ruby on Rails

    Un desarrollador Ruby on Rails es un programador especializado en el framework Ruby on Rails, una herramienta muy popular para el desarrollo de aplicaciones web.


  • Desarrollo de aplicaciones web: Creación de aplicaciones web personalizadas.
  • Integración de APIs: Conexión de aplicaciones con otros servicios.
  • Mantenimiento de aplicaciones: Actualización y resolución de problemas en aplicaciones existentes.
